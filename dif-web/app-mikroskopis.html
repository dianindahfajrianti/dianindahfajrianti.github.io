App Mikroskopis

Project information

  • Category : Web Development
  • Project date : 2023
  • Prototype : https://s.id/aplikasicuti-prototype
  • Github : https://github.com/dianindahfajrianti/aplikasicutiii

App Mikroskopis merupakan salah satu projek desain freelance. Dimana, saya membuat desain user interface App Mikroskopis ini menggunakan figma.

App Mikroskopis dibuat untuk melihat informasi mengenai mikroskopis. Fitur yang terdapat pada App Mikroskopis ini adalah login, registrasi, forget password dan search bar filter.

Setiap konten yang ada disusun mulai dari mencari dan mendefinisikan konten apa saja yang nantinya akan ada dalam aplikasi sampai ke tahap membuat evaluasi rancangan desain. Ada 4 menu yang menjadi lingkup kerja pada projek ini yaitu :

  1. Menu Mikroskopik untuk urin
  2. Menu Mikroskopik untuk feses
  3. Menu Quiz
  4. Menu Profile
