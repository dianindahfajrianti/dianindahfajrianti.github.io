Ticketing

Project information

  • Category : Web Development
  • Project date : 2022
  • Prototype : https://s.id/ticketing-prototype
  • Github : https://github.com/dianindahfajrianti/ticketing

Ticketing ini di buat untuk sistem kiosk IT Clinic Axioo, agar memudahkan proses antrian pelanggan sesuai dengan keperluan pelanggan. Saya membuat user interface Survey Pelanggan ini menggunakan HTML, CSS dan Javascript dengan menggunakan Bootstrap 5.

Berikut merupakan kebutuhan fungsional dan kebutuhan antarmuka pada Ticketing :

  1. Menggunakan background video yang jika ingin menyetak antrian dapat mengklik layar dan terdapat pilihan layanan sesuai dengan kebutuhan pelanggan.
  2. Terdapat 4 pilihan layanan yaitu servis, pengambilan servis, pembelian sparepart dan lain-lain.
  3. Setelah memilih layanan sistem akan otomatis mencetak nomor antrian sesuai dengan layanan yang di pilih.
