Portal Web

Project information

  • Category : Web Development
  • Project date : 2022
  • Prototype : https://s.id/portalweb-prototype
  • Github : https://github.com/dianindahfajrianti/portal-web

Portal web ini merupakan penyedia layanan informasi, dimana dibuat untuk mengarahkan pengguna agar dapat mudah mengakses kumpulan website dari sistem PT Indo Mega Vision yang sudah dibuat. Saya membuat user interface portal web ini menggunakan HTML, CSS dan Javascript dengan menggunakan framework Bootstrap 5.

Berikut merupakan kebutuhan fungsional dan kebutuhan antarmuka pada Portal Web :

  1. Terdapat beberapa card dengan informasi nama sistem dan tanggal update sistem yang jika di klik akan mengarahkan ke sistem tersebut.
  2. Jika ingin mengakses beberapa sistem sekaligus, dapat login melalui portal web.
