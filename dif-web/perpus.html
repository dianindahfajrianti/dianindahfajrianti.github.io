Perpustakaan Online merupakan salah satu projek yang di berikan perusahaan untuk saya kerjakan. Dimana, jobdesk saya yaitu membuat desain UI/UX dan mengaplikasikannya dalam HTML, CSS dan Javascript.

Untuk membuat desain user interface dan prototype saya menggunakan Figma dan dalam mengaplikasikannya dalam HTML, CSS dan JavaScript saya menggunakan framework bootstrap 5. Untuk bagian Admin CMS saya menggunakan template AdminLTE dengan framework bootstrap 4.
