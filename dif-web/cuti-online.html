Projek aplikasi cuti online ini merupakan sebuah projek membuat website untuk cuti online pegawai. Projek ini diberikan oleh tempat magang yang pernah saya ikuti.

Aplikasi cuti online ini dibuat menggunakan Bahasa pemrograman php dengan database phpMyAdmin dan ditambahkan framework bootstrap.

Berikut merupakan kebutuhan fungsional dan kebutuhan antarmuka pada Aplikasi Cuti Online :

  1. Memiliki 4 hak akses yaitu Admin (untuk HRD), Kasi, Kadep, Pegawai.
  2. Login Admin, Kasi, Kadep, Pegawai.
  3. Beranda Admin, Kasi, Kadep, Pegawai.
  4. Tambah, lihat, edit, hapus Data Karyawan.
  5. Tambah, lihat, cetak Approval data karyawan yang ingin mengajukan cuti.
  6. Pencarian data cuti dan cetak laporan data.
  7. Pengajuan Cuti Pegawai.
