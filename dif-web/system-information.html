System Information

Project information

  • Category : Web Development
  • Project date : 2023
  • Prototype : https://s.id/systeminformation-prototype
  • Github : https://github.com/dianindahfajrianti/System_Information

Website System Information ini digunakan untuk mengetahui informasi dari sistem suatu komputer ataupun laptop seperti processor, CPU, memory dan informasi sistem lainnya. Dengan adanya website system information ini, user dapat dengan mudah mengetahui spesifikasi komputer atau laptop dengan lebih mudah.

Saya membuat user interface Website System Information ini menggunakan HTML, CSS dan Javascript dengan menggunakan framework Bootstrap 5 dan nodejs.
